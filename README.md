# HR Workforce Analytics Dashboard

This project is a Power BI dashboard created for analyzing HR data such as employee attrition, salary distribution, age groups, job satisfaction, and workforce trends.

## Tools Used
- Power BI Desktop
- Excel / CSV Dataset
- Power Query
- DAX

## Dashboard Features
- Total employee count
- Attrition count and attrition rate
- Average age and salary
- Attrition by department
- Attrition by age group
- Job satisfaction analysis
- Work-life balance insights

## Files Included
- Power BI dashboard file (.pbix)
- Dashboard screenshot
- Dataset file
